# 소프트웨어공학 - SW 설계˙디자인 프로젝트

## 💁 About Project
> '소프트웨어공학' 수업에서 진행한 프로젝트입니다.
> 본 프로젝트에서는 3명의 팀원들과 소프트웨어 설계 과정을 실습하였습니다.
> 주제는 "의류 쇼핑 사이트"를 입니다.
> 각 회원이 의류 상품을 등록해서 판매할 수 있고, 다른 회원이 등록한 의류상품을 구매할 수 있다고 가정하였습니다.
> 처음 요구사항 분석부터 시작하여 각종 디자인 산출물들을 바탕으로 C++ 코드를 구현하였습니다.
